# Java DSA Practice

This repository contains **Java implementations of Data Structures and Algorithms (DSA)** along with short reference notes.  
It is created for practice, revision, and strengthening problem-solving skills in Java.

## Topics Covered

- Arrays  
- ArrayList  
- Linear Search  
- Binary Search  
- Sorting Algorithms  
- Strings  
- StringBuilder  
- Recursion  
- Basic DSA concepts and notes for revision  

## Repository Structure

- `src/` – Java source files containing DSA problems, solutions, and notes  
- `nbproject/` – NetBeans project configuration files  
- `build.xml`, `manifest.mf` – Build and project metadata  

## How to Run

### Using IDE
- Open the project in **NetBeans / VS Code / Eclipse**
- Navigate to the `src` folder
- Run the required Java class

### Using Command Line
```bash
javac src/FileName.java
java FileName
```
